Installation
============

pgmpy supports Python >= 3.10.

PyPI
----

.. code-block:: bash

   pip install pgmpy

Conda
-----

.. code-block:: bash

   conda install -c conda-forge pgmpy

Development Version
-------------------

To install the latest ``dev`` branch from GitHub:

.. code-block:: bash

   pip install git+https://github.com/pgmpy/pgmpy.git@dev
